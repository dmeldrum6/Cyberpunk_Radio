About US Radio

US Radio is a cyberpunk themed internet radio streaming application that lets you discover and listen to thousands of radio stations from across the United States.

Stream your favorite music, news, sports, and talk radio stations anywhere, anytime. The app uses the Radio Browser API to provide you with a comprehensive collection of stations organized by genre.

Features

  • Stream thousands of US radio stations
  • Browse stations by genre
  • Search for specific stations
  • Background playback
  • Cyberpunk user interface
US Radio v1.0.0
Powered by Radio Browser API

How to Use

  • Select a genre from the home screen
  • Browse stations within that genre
  • Tap on a station to start playback
  • Use the player controls at the bottom to pause/play
  • Return to genres by tapping the back button

Troubleshooting

  • If a station won't play, try another one (some streams may be temporarily offline)
  • Check your internet connection if you experience buffering
  • For best performance, use on WiFi networks
  • Restart the app if you encounter persistent issues

Permissions

This app requires the following permissions:

  • Internet: To stream radio stations
  • Wake lock: To allow background playback
